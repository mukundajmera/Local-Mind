## Features

### Sources Panel (Left Sidebar)
- **Upload documents**: PDF, Markdown (.md), and TXT files supported
- **Checkbox selection**: Select multiple sources to include in chat context
- **Click to preview**: Click source title to view Summary and Key Topics
- **Delete sources**: Hover and click X to remove

### Workspace (Center Panel)

#### Source Guide View
When you click on a source title, you'll see:
- **Summary**: AI-generated overview of the document
- **Key Topics**: Main themes extracted from the content
- **Suggested Questions**: Starting points for exploration

#### Chat View
- **Multi-source chat**: Chat queries selected sources (checked boxes)
- **Prompt suggestions**: Quick-start buttons for common queries
- **Pin responses**: Save valuable AI insights to Notes

### Notes Panel (Right Sidebar)
- Toggle with the notes icon in the header
- **Add notes**: Quick capture of thoughts
- **Pinned messages**: AI responses you've saved
- **Pin/Unpin**: Organize important content

---

## Tips

1. **Select before chatting**: Check the boxes next to sources you want to include in your query
2. **Use the Help button** (`?` in header) for a quick workflow reminder
3. **Pin important findings** so you don't lose them
4. **Theme toggle**: Switch between dark and light themes in the header
